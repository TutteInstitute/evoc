Getting Started Notebooks
--------------------------

These notebooks provide hands-on introduction to EVoC:

* **Basic Clustering Example**: Introduction to EVoC with synthetic data
* **Advanced Parameter Tuning**: Guide to selecting optimal parameters  
* **Hierarchical Analysis**: Working with multi-layer clustering results

Specialized Use Cases
---------------------

* **Embedding Type Comparison**: Clustering different embedding formats
* **Performance Benchmarks**: Speed and quality comparisons

Running the Notebooks
----------------------

To run these notebooks locally:

1. Install EVoC and Jupyter: ``pip install evoc jupyter``
2. Clone the repository: ``git clone https://github.com/TutteInstitute/evoc.git``  
3. Navigate to notebooks: ``cd evoc/doc/source/notebooks``
4. Start Jupyter: ``jupyter notebook``

All notebooks are also viewable directly in the documentation without requiring a local installation.
